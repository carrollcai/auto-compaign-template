--- 
title: 营销活动
---

主要利用webpack实现功能
### init
```
NAME="filename" npm run init
```
- mkdir "filename"
- copy template
- index.html add static source path

### dev
```
NAME="filename" npm run dev
```
- start webpack-dev-middleware
- set proxy

### build
```
NAME="filename" npm run build
```
- babel es6 convert es5, postcssgit init..
- package
- compress
